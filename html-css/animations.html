CSS ANIMATIONS

На главную страницу

Есть 9 свойств, которые позволяют контролировать CSS animations:
• animation-name;
• animation-duration;
• animation-timing-function;
• animation-delay;
• animation-iteration-count;
• animation-direction;
• animation-play-state;
• animation-fill-mode;
• animation;

В этом примере:
my-animation - имя анимации (animation-name).
2s - длительность анимации (animation-duration).
ease-in - функция времени (animation-timing-function).
1s - задержка перед началом анимации (animation-delay).
3 - количество итераций анимации (animation-iteration-count).
alternate - направление анимации (animation-direction).
forwards - режим заполнения (animation-fill-mode).
running - состояние анимации (animation-play-state).
