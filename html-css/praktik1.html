Тег <br> - Перенос текста на другую строку (разрыв текста).

Этот тег не имеет закрывающего тега, так как он не ограничивает блок текста, а служит лишь для указания браузеру, что после него следует начать новую строку текста.

Пример использования тега <br>:

Вот тут был использован тег <br>

Тег <html>

Это основной тег, с которого начинается каждый HTML-документ. Он определяет начало и конец HTML-документа и охватывает всё содержание веб-страницы.

Пример использования тега <html>:
<html>
<!-- Весь контент страницы идет здесь -->
</html>

Тег <title>

Это основной тег <title> в HTML-документе. Он используется для задания заголовка страницы в браузере и является важным элементом для SEO (оптимизации для поисковых систем).

Пример использования тега <title>:
<title> Заголовок вашей страницы </title>

Тег <head>

Это секция <head> в HTML-документе. Она содержит метаданные и другую информацию, которая не отображается напрямую на странице, но важна для браузера и поисковых систем.

Пример использования секции <head>:
<head>
<title>Заголовок вашей страницы</title>
<meta charset="UTF-8">
<link rel="stylesheet" href="styles.css">
</head>

Тег <body>

Это секция <body> в HTML-документе. Она содержит видимое содержание веб-страницы, такое как текст, изображения, ссылки, формы и другие элементы, которые отображаются в браузере пользователя.

Пример использования секции <body>:
<body>
<h1>Привет, мир!</h1>
<p>Это пример веб-страницы.</p>
</body>

Тег <p>

Тег <p> используется для создания абзацев в HTML. Он обозначает текстовый параграф и обычно отделяет его от других элементов на странице.

Пример использования тега <p>:
<p>Это пример текстового параграфа. Абзацы полезны для разделения и структурирования текста.</p>
<p>Еще один абзац текста.</p>
</p>
