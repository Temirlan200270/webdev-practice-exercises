Задача #1

С помощью метода splice преобразуйте массив в следующий:
[1, 4, 5]

Решение:
<script>
var arr1 = [1, 2, 3, 4, 5];
arr1.splice(1, 2);
alert("arr1);
</script>

Задача #2

С помощью метода splice сделайте из него массив:
[1, 2, 3, 'a', 'b', 'c', 4, 5]

Решение:
<script>
var arr2 = [1, 2, 3, 4, 5];
arr2.splice(3, 0, 'a', 'b', 'c');
alert("arr2);
</script>

Задача #3

С помощью метода splice сделайте из него массив:
[1, 'a', 'b', 2, 3, 4, 'c', 5, 'e']

Решение:
<script>
var arr3 = [1, 2, 3, 4, 5];
arr3.splice(1, 0, 'a', 'b'); arr3.splice(6, 0, 'c'); arr3.splice(8, 0, 'e');
alert("arr3);
</script>

Задача #4

Сделайте из этого массива следующий:
[1, 2, 3]

Решение:
<script>
var arr4 = [1, 2, 3, 4, 5];
arr4 = arr4.slice(0, 3);
alert("arr4);
</script>

Задача #5

Сделайте из этого массива следующий:
[4, 5]

Решение:
<script>
var arr5 = [1, 2, 3, 4, 5];
var newArr = arr5.slice(3);
alert("newArr);
</script>
