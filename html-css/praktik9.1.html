Задача #1

С помощью цикла сформируйте строку, заполненную 5-тью дефисами.

Решение:
<script>
var result1 = '';
for (var i = 0; i < 5; i++) {
    result1 += '-';
}
alert("result1);
</script>

Задача #2

С помощью цикла сформируйте строку '123456789'.

Решение:
<script>
var result2 = '';
for (var i = 1; i <= 9; i++) {
    result2 += i;
}
alert("result2);
</script>

Задача #3

С помощью цикла сформируйте строку '987654321'.

Решение:
<script>
var result3 = '';
for (var i = 9; i >= 1; i--) {
    result3 += i;
}
alert("result3);
</script>

Задача #4

С помощью цикла сформируйте строку '-1-2-3-4-5-6-7-8-9-'.

Решение:
<script>
var result4 = '-';
for (var i = 1; i <= 9; i++) {
    result4 += i + '-';
}
alert("result4);
</script>
