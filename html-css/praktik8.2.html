Обращение к символам строки

Задача 1

Создайте переменную str и присвойте ей значение 'abcde'. Обращаясь к отдельным символам этой строки выведите на экран символ 'a', символ 'c', символ 'e'.

Решение:
<script>
var str = 'abcde';
alert("Символ 'a': " + str[0]);
alert("Символ 'c': " + str[2]);
alert("Символ 'e': " + str[4]);
</script>

Задача 2

Создайте переменную num и присвойте ей значение '12345'. Найдите произведение (умножение) цифр этого числа.

Решение:
<script>
var num = '12345';
var product = 1;
for (var i = 0; i < num.length; i++) {
product *= parseInt(num[i]);
}
alert("Произведение цифр: " + product);
</script>

Практика

Задача 3

Считаем количество секунд в часе, в сутках, в месяце.

Решение:
<script>
var secondsInHour = 60 * 60;
var secondsInDay = secondsInHour * 24;
var secondsInMonth = secondsInDay * 30; // предполагаем, что месяц состоит из 30 дней
alert("Секунд в часе - " + secondsInHour + ", в сутках - " + secondsInDay + ", в месяце - " + secondsInMonth);
</script>

Задача 4

Выводим текущее время в формате 'час:минута:секунда'.

Решение:
<script>
var now = new Date();
var hour = now.getHours();
var minute = now.getMinutes();
var second = now.getSeconds();
alert("Текущее время - " + hour + ":" + minute + ":" + second);
</script>

Задача 5

Возводим число в квадрат.

Решение:
<script>
var number = 5;
var square = number ** 2; // возведение в квадрат
alert("Квадрат числа " + number + " равен " + square);
</script>

ПРОДОЛЖЕНИЕ ЭТОЙ ЖЕ ПРАКТИКИ

Работа с присваиванием и декрементами

Задача 6

Переделайте этот код так, чтобы в нем использовались операции +=, -=, *=, /=. Количество строк кода при этом не должно измениться.

Задача
var num = 47;
num = num + 7;
num = num - 18;
num = num * 10;
num = num / 15;
alert(num);

Решение:
<script>
var num = 47;
num += 7; // увеличиваем на 7
num -= 18; // уменьшаем на 18
num *= 10; // умножаем на 10
num /= 15; // делим на 15
alert("Результат вычислений: " + num);
</script>

Задача 7

Переделайте этот код так, чтобы в нем использовались операции ++ и --. Количество строк кода при этом не должно измениться.

Задача:
var num = 10;
num = num + 1;
num = num + 1;
num = num - 1;
alert(num);

Решение:
<script>
var num = 10;
num++; // увеличиваем на 1
num++; // увеличиваем на 1
num--; // уменьшаем на 1
alert("Результат вычислений: " + num);
</script>
