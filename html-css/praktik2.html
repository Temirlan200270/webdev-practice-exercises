Тег <h1>

Тег <h1> представляет собой самый высокий уровень заголовка в HTML. Он используется для выделения основного заголовка документа.

Пример использования тега <h1>:

Тег <br>

Тег <br> используется для создания разрыва в тексте и перехода на новую строку без создания абзаца.

Пример использования тега <br>:

                    Этот текст будет
разделен на две строки.

Тег <i>

Тег <i> используется для выделения текста курсивом. Он обычно применяется для слов или фраз, чтобы указать, что они имеют курсивное начертание.

Пример использования тега <i>:

                    Этот текст в <i>курсиве</i>.

Тег <tt>

Тег <tt> используется для отображения текста моноширинного шрифта, как если бы это был машинописный текст. Он обычно используется для отображения кода.

Пример использования тега <tt>:

                    <tt>Этот текст моноширинный</tt>.

Тег <pre>

Тег <pre> используется для отображения предварительно отформатированного текста с сохранением всех пробелов и переносов строк, как они записаны в исходном коде.

Пример использования тега <pre>:

                    <pre>
                        Этот текст
                        сохраняет пробелы и
                        переносы строк.
                    </pre>

Тег <u>

Тег <u> используется для подчеркивания текста. Он обычно указывает на то, что текст является подчеркнутым.

Пример использования тега <u>:

                    <u>Этот текст подчеркнут</u>

Тег <strike> (или <s>)

Тег <strike> (или <s>) используется для перечеркивания текста. Он обычно указывает на то, что текст был удален или не актуален.

Пример использования тега <strike> (или <s>):

                    <strike>Этот текст перечеркнут</strike>

Тег <big>

Тег <big> используется для увеличения размера текста. Он придает тексту больший размер, чем остальной текст на странице.

Пример использования тега <big>:

                    <big>Этот текст большего размера</big>

Тег <small>

Тег <small> используется для уменьшения размера текста. Он придает тексту меньший размер, чем остальной текст на странице.

Пример использования тега <small>:

                    <small>Этот текст меньшего размера</small>

Тег <sup>

Тег <sup> используется для отображения текста в верхней части строки.

Пример использования тега <sup>:

                    H2O

Тег <sub>

Тег <sub> используется для отображения текста в нижней части строки.

Пример использования тега <sub>:

                    E=mc2
